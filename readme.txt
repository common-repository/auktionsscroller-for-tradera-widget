Visa dina auktioner från Tradera på din blogg.

== Description ==

Visa dina auktioner från Tradera på din blogg. Med hjälp av auktionsscroller.se och Traderas API uppdateras auktionerna en gång per dygn.

Gör så här:

1.  Leta reda på ditt ftg-nummer, lättast hittar du det i adressfältet på din egen omdömessida.
2.  Skriv in ditt ftg-nummer och ditt alias i widgetens inställningar.
3.  Ändra övriga inställningar som vertikal eller horisontell scrollning, antal auktioner som ska visas och om scrollerna ska snurra eller inte.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then

activate the Plugin from Plugins page.

Packa upp zip-filen och lägg innehållet i **/wp-content/plugins/** -katalogen som du hittar där wordpress är installerat.

Aktivera pluginet på plugins-sidan och lägg till widgeten till någon av dina sidebars.
